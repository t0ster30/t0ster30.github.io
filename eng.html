conditionals table

conditional where how
zero facts, general habits if + subj + present simple verb // subject + present simple
first possible future or present if + subj + present simple verb // subject + will\won't\may\might + infinitive
second imaginary present or future if + subj + past simple // subject + would\wouldn't + infinitive
third imaginary past if + subj + had\hadn't + past participle // subject + would\wouldn't have + past participle

so it looks like this:

zero:if he plays he doesn't sting.
first:if we wait we will get the presents.
second:if horses could fly, they would use wings.
third:if he hadn't sold the onions we would have made a salad.
