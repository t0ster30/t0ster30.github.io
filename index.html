mt *roff macro

Essentially it is the -en macro from Neatroff.

To see what this macro can do use grep -- tmac.t.

You need to run

sed 's/AUTHOR/[insert your name here]/g'

and

sed 's/INSTITUTION/[your place of education]/g'

before you start using the macro.

Also don't forget to mount the fonts.

.ft - R  [your Regular font]
.ft - B  [your Bold font]
.ft - I  [your Itali font]
.ft - CR [your Code-Regular font]
